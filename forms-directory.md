# 🗺️ Répertoire Complet des Boutons et Liens
Ce document est la carte routière de votre application. Il recense tous les principaux boutons de navigation et d'action, leur emplacement et leur destination.
---
## 1. Navigation Principale (Header & Footer)
Ces liens sont présents sur la majorité des pages.
| Texte du Bouton/Lien | Emplacement | Destination | Rôle |
|---|---|---|---|
| `Nos Spécialités` | Header | `/#professions` | Fait défiler la page d'accueil vers la section des métiers. |
| `Blog` | Header / Footer | `/blog` | Mène à la page principale du blog. |
| `Clic+Cockpit` | Header / Footer | `/internal` | Accès au portail interne pour les courtiers. |
| `Contact` | Header / Footer | `/comment-ca-marche` | Mène à la page "Comment ça marche" qui contient les infos de contact. |
| `Soumission chantier en 2h*` | Header | `/soumission/chantier-rapide`| Lance le quiz d'éligibilité pour la soumission rapide. |
| `Plan Directeur` | Footer | `/internal/information-architecture` | Affiche le document sur l'architecture de l'information. |
| `Cyber-Risques` | Footer | `/assurance-cyber-risques` | Mène à la page d'information sur l'assurance Cyber. |
| `Expérimental` | Footer | `/experimental` | Accède à la page de test pour les nouveaux composants. |
| `Portail des Professionnels` | Footer | `/assurance-pros` | Mène au portail de ressources pour les professionnels de l'assurance. |

## 2. Pages Publiques
Boutons accessibles par tous les visiteurs.
| Texte du Bouton/Lien | Emplacement | Destination | Rôle |
|---|---|---|---|
| `Soumission Clic+Express` | Page d'accueil | `/soumission/express` | Ouvre l'outil de soumission rapide par IA. |
| `Explorer les profils` | Page d'accueil | `/#professions` | Fait défiler vers la section des métiers. |
| `Toutes nos soumissions` | Page d'accueil | `/soumission` | Mène à la page listant toutes les options de soumission. |
| `Un Clic+Assur soumission` | Cartes de profil (Carrousel) | `/family/[...]/[...]/soumission` | Mène à la page de soumission spécifique au profil métier. |
| `Commencer la soumission` | Page `Comment ça marche` | `/#professions` | Fait défiler la page d'accueil vers la section des métiers. |
| `Lancer le Quiz` | CTA `Quiz : Rôle sur le chantier` | `/outils/quiz-role-construction` | Démarre le quiz pour déterminer le rôle de l'utilisateur. |
| `Lancer le Calculateur` | CTA `Calculateur de Coûts Indirects` | `/outils/calculateur-couts-indirects` | Ouvre l'outil de calcul des "soft costs". |
| `Lire l'article` | Cartes du Blog | `/blog/[slug]` | Ouvre la page de l'article de blog correspondant. |

## 3. Clic+Cockpit (Portail Interne)
Boutons accessibles uniquement depuis le portail interne.
| Texte du Bouton/Lien | Emplacement | Destination | Rôle |
|---|---|---|---|
| `Créer une Fiche Nominative` | Dashboard Interne | `/internal/fiche-nominative` | Ouvre l'outil de création et de fusion de données client. |
| `OUVRIR MON MARKETER` | Dashboard Interne | `/internal/mon-marketer` | Accède au cockpit de propulsion des soumissions. |
| `Ouvrir le Workflow Complet` | Dashboard Interne (Onglets) | `/internal/[workflow]` | Ouvre la page dédiée au workflow (Nouvelle Affaire, Modifications, etc.). |
| `MON MARKETER (Propulsion)` | Actions Rapides (Pop-up) | `/internal/mon-marketer` | Raccourci vers le cockpit de mise en marché. |
| `Portail Automobile` | Actions Rapides (Pop-up) | `/assurance-automobile/transactions` | Ouvre la page des générateurs de documents auto. |
| `Chantier & Construction` | Actions Rapides (Pop-up) | `/assurance-construction/transactions` | Ouvre la page des générateurs de documents construction. |
| `Voir Annexe Technique` | Laboratoire Jotform | `/experimental/jotform/[id]` | Affiche les détails techniques d'un bloc de formulaire. |
| `Ouvrir le Formulaire Jotform` | Laboratoire Jotform | `https://form.jotform.com/[id]` | Ouvre le formulaire JotForm directement dans un nouvel onglet. |
| `PROPULSER VERS [Assureur]` | `Mon Marketer` | `https://form.jotform.com/[id]?data...` | Génère une URL pré-remplie et l'ouvre pour finaliser la proposition. |
